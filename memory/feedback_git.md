---
name: Nunca commitar sem solicitação
description: Nunca executar git commit, git add ou git push sem pedido explícito do usuário
type: feedback
---

Nunca executar `git commit`, `git add` ou `git push` sem que o usuário peça explicitamente.

**Why:** O usuário prefere controlar todos os commits manualmente e rejeitou tentativas de commit automático em múltiplas ocasiões.

**How to apply:** Criar/editar arquivos normalmente, mas nunca encadear com git add/commit/push. Deixar o usuário commitar quando quiser.
