## Reportando uma Vulnerabilidade

Utilize o canal de **GitHub Security Advisories** para reportar vulnerabilidades de forma privada:

1. Acesse a aba **Security** do repositório
2. Clique em **Report a vulnerability**
3. Descreva o problema com o máximo de detalhes possível (passos para reproduzir, impacto esperado, versão afetada)

Você receberá uma primeira resposta em até **72 horas**.

Após a análise:
- Se a vulnerabilidade for confirmada, trabalharemos em um fix e publicaremos um advisory assim que a correção estiver disponível
- Se for descartada, explicaremos o motivo na própria thread do report

Por favor, não abra issues públicas para vulnerabilidades de segurança.
